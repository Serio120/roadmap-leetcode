## Qué es LeetCode
LeetCode es una plataforma en línea con cientos de problemas de programación utilizada por ingenieros de software para prepararse para entrevistas técnicas. Empresas como Google, Meta, Amazon y Microsoft utilizan problemas similares en su proceso de contratación para evaluar cómo los candidatos abordan los desafíos algorítmicos. No es necesario resolver miles de problemas para estar preparado. Lo que importa es comprender los patrones detrás de los problemas lo suficientemente bien como para aplicarlos a aquellos que nunca has visto antes. LeetCode es el terreno de práctica, no el objetivo.

## Elija un lenguaje
Para LeetCode y entrevistas técnicas, el lenguaje que utilice importa menos que el conocimiento que tenga de él. Elija un lenguaje y manténgase con él durante toda su preparación. Cambiar entre lenguajes desperdicia tiempo y dispersa su enfoque. Lo que realmente les importa a los entrevistadores es si puede escribir código limpio y correcto y explicar su razonamiento de manera clara. Dicho esto, algunos lenguajes tienen ventajas prácticas: Python es conciso y rápido de escribir, lo que resulta útil bajo presión de tiempo. Java y C++ son comunes en empresas que valoran el rendimiento. JavaScript es una elección natural si proviene del desarrollo frontend.
