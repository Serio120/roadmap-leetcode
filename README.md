# Qué es LeetCode
LeetCode es una plataforma en línea con cientos de problemas de programación utilizada por ingenieros de software para prepararse para entrevistas técnicas. Empresas como Google, Meta, Amazon y Microsoft utilizan problemas similares en su proceso de contratación para evaluar cómo los candidatos abordan los desafíos algorítmicos. No es necesario resolver miles de problemas para estar preparado. Lo que importa es comprender los patrones detrás de los problemas lo suficientemente bien como para aplicarlos a aquellos que nunca has visto antes. LeetCode es el terreno de práctica, no el objetivo.
